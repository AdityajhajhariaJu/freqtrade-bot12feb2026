# bot13feb2026

Synced project files for the multi-strategy Binance futures bot.

## Core files
- `trade_loop.py` — live execution loop
- `strategies.py` — strategy engine and filters
- `new_strategies_2h.py` / `new_strategies_4h.py` — higher-timeframe strategy packs
- `reports/` — logs, diagnostics, and trade reports

## Security note
Runtime secret config and virtual environment files are excluded from git.
